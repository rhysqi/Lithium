# Lithium
Text editor that using native X11. Suitable in Unix/Unix-like environment. <br>
Good for C/C++ and Assembly development.

# Features
- Syntax Highlighting
- File format detection
- Custom Themes
- Snippets
- LLVM LSP

# Tested Platform
- [x] FreeBSD 12
- [x] LinuxMint 21

# Dependencies & Toolchains
- libX11
- LLVM v13
- CMake v3.2

# Build
```
mkdir build
cd build
cmake ..
make
```
